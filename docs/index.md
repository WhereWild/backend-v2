# WhereWild Docs

Welcome to the WhereWild backend docs. These pages cover the core libraries and
one-off scripts that power data processing and API behavior.

## What’s Here

- **Libraries**: reusable helpers in `util/` (GIS lookup, taxonomy traversal,
  ranking utilities).
- **Scripts**: occasional data workflows in `scripts/` (build, enrich, and
  indexing tasks).

For more information, see the README on the left.
